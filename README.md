# Automated Resume Screening

AI-powered resume screening and candidate ranking system using Natural Language Processing (NLP) and Machine Learning.

## Overview

Recruiters often spend significant time manually reviewing resumes. This project automates the screening process by analyzing resumes, extracting relevant information, and ranking candidates against a job description.

## Features

- Resume text extraction
- Data preprocessing
- Skill matching
- Candidate ranking
- Similarity scoring
- Automated screening workflow

## Tech Stack

- Python
- Pandas
- NumPy
- Scikit-Learn
- NLTK / SpaCy
- Jupyter Notebook

## Workflow

1. Upload resumes
2. Extract text
3. Preprocess data
4. Generate features
5. Compare with job description
6. Compute similarity scores
7. Rank candidates
